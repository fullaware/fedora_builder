fedora_builder
==============

fedora 20, 21 apps and tweaks for HP 9470m


Add:
`acpi_osi=Linux acpi_backlight=vendor`  
to:
`/etc/default/grub (the line GRUBCMDLINELINUX=" ..... "`  
then:
`grub2-mkconfig -o /boot/grub2/grub.cfg`
